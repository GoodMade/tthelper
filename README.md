
TapTop Helper - расширение для Chrome, которое делает работу в редакторе TapTop удобнее, быстрее и спокойнее.
Главная идея TapTop Helper простая: меньше кликов, меньше раздражающих ограничений интерфейса, больше контроля над рабочим пространством.

# TapTop Helper

TapTop Helper — расширение для Chrome, которое добавляет удобные функции в редактор [taptop.pro](https://taptop.pro): поиск по слоям, улучшенный редактор кода, импорт/экспорт слоёв, мини-браузер и другие быстрые инструменты.

## Возможности

### Общие

- Работает на страницах редактора Taptop: `/-/cms`.
- Все функции включаются и выключаются в popup расширения.
- Настройки сохраняются в Chrome.
- Часть функций применяется сразу, а часть требует перезагрузки страницы — расширение покажет кнопку «Применить».

### Редактор кода

- Поиск в Ace-редакторе по `Cmd+F` / `Ctrl+F`.
- Изменение размера окна редактора кода.
- Кнопка растягивания окна редактора по высоте.
- Увеличенная рабочая область кода внутри модального окна.
- Отключение AI Code Generator в редакторе.
- Временные вкладки для нескольких вариантов кода.

### Слои

- Поиск по слоям в левой панели.
- Подсветка найденных слоёв.
- Переход между совпадениями через `Enter` и `Shift+Enter`.
- Сохранение частых поисковых запросов как быстрых тегов.
- Кнопка «Редактировать» рядом с выбранным embed-слоем.
- Виджет `Script` в панели виджетов. (Embed слой с подготовленными тегами script и style)
- Быстрые кнопки скрытия выбранного слоя:
  - скрыть/показать слой через display:none(добавляет или убирает класс helper--d-none) или полностью;   
- Копирование слоёв между разными проектами Taptop.
  (есть Обработка конфликтов CSS-классов при вставке слоя.)
- Экспорт выбранного слоя в JSON.
- Импорт слоя из JSON в буфер Taptop с последующей вставкой через `Cmd+V` / `Ctrl+V`.

### Патчи

- Отключение автозакрытия слоёв, скрытых через `display:none` или настройку видимости.

### Правая панель

- Отключение всплывающих подсказок. 
  (Временный показ подсказок при зажатом `Ctrl`.)
- Дополнительные единицы измерения:
  - `dvh`
  - `svh`
  - `lvh`
  - `vmin`
  - `vmax`
  - `ch`
- Поиск и замена текста на холсте.

### Мини-браузер

- Мини-браузер прямо внутри редактора.
- Закреплённые вкладки для быстрых ссылок.
- Настройка списка вкладок в popup:
  - добавить ссылку;
  - удалить ссылку;
  - включить или выключить ссылку;
  - поменять порядок.
- Адресная строка.
- История открытых страниц.
- Подсказки из истории и закреплённых ссылок.
- Закладки с папками.
- Поиск по закладкам.
- Открытие текущей страницы:
  - в новой вкладке Chrome;
  - в отдельном окне;
  - в Chrome Side Panel.
- Закрепление мини-браузера на экране.
- Изменение ширины панели.
- Режимы устройств для проверки адаптива:
  - iPhone SE;
  - iPhone 12 Pro;
  - iPhone 14 Pro Max;
  - Pixel 7;
  - iPad Mini;
  - iPad Air;
  - iPad Pro;
  - Full HD.
- Подгонка ширины панели под выбранное устройство.

### Верхняя панель

- Кнопка открытия текущего сайта в новой вкладке.
  (Открывается корневой адрес сайта, который сейчас редактируется)

### Предпросмотр

- Увеличение ширины режима предпросмотра.
- Скрытие горизонтального скроллбара предпросмотра.



### 📦 Как установить расширение:

1. Перейдите во вкладку «Releases» справа и скачайте архив последней версии.
2. Распакуйте скачанный архив в любую удобную папку.
3. Откройте в Chrome страницу: chrome://extensions
4. Включите «Режим разработчика» в правом верхнем углу.
5. Перетащите распакованную папку мышкой прямо в окно браузера.

⚠️ Важно: Не удаляйте и не перемещайте папку после установки, Chrome запускает расширение напрямую из неё.

🔄 Как обновить расширение:
Просто скачайте новый архив из релизов, распакуйте его поверх старой папки с заменой файлов и нажмите иконку «Обновить» (круговая стрелка) на странице chrome://extensions.
